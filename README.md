# Hotel-Booking-System
This is a personal project, main focus is to learn and showcase skills like .NET Backend Development, System Design, AWS Deployment and DevOps. 

## Tech Stack and Tools:
- **Backend** : .NET 8 Web API
- **Database** : PostgreSQL
- **Containerization** : Docker
- **Deployment** : AWS Cloud Service
- **Version Control** : Git & GitHub
- **CI/CD** : GitHub Action
